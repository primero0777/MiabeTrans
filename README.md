# MiabeTrans

## Installation
To install MiabeTrans, follow these steps:
1. Clone the repository:
   ```sh
   git clone https://github.com/primero0777/MiabeTrans.git
   ```
2. Navigate to the project directory:
   ```sh
   cd MiabeTrans
   ```
3. Install the required dependencies:
   ```sh
   npm install
   ```

## Features
- **User-friendly Interface**: Navigate easily through the application.
- **Fast Performance**: Optimized for speed and efficiency.
- **Secure Transactions**: All transactions are secure and encrypted.
- **Multilingual Support**: Available in multiple languages to cater to a wider audience.
